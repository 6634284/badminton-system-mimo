# 会员导入模板说明

## Excel模板格式

| 列名 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 姓名 | 是 | 会员真实姓名 | 张三 |
| 手机号 | 是 | 11位手机号，自动去重 | 13800138001 |
| 性别 | 否 | 0=未知, 1=男, 2=女 | 1 |
| 会员卡号 | 否 | 留空则自动生成 | M00001 |
| 等级 | 否 | 默认0，数值 | 1 |
| 来源 | 否 | direct/walk_in/referral/online | direct |
| 备注 | 否 | 备注信息 | 老会员 |

## 注意事项

1. **手机号去重**：系统自动检测重复手机号，跳过已存在的会员
2. **敏感数据加密**：姓名、手机号在数据库中加密存储
3. **异步处理**：导入任务异步执行，大文件（1000+行）不会阻塞
4. **进度查询**：上传后返回任务ID，可轮询查看导入进度
5. **错误报告**：导入完成后返回错误行详情（行号+原因）

## 导入流程

```
上传Excel → 解析验证 → 异步导入(BullMQ) → 进度更新(Redis) → 完成通知
```

## 错误码说明

| 错误 | 说明 |
|------|------|
| DUPLICATE_PHONE | 手机号已存在，跳过 |
| INVALID_PHONE | 手机号格式错误 |
| MISSING_REQUIRED | 缺少必填字段（姓名或手机号） |
| INVALID_GENDER | 性别值不是0/1/2 |
