# 管理员操作手册

## 1. 首次登录

1. 访问管理后台：`https://admin.example.com`
2. 使用管理员账号登录（平台管理员 / 俱乐部管理员）
3. 首次登录后请立即修改密码

## 2. 俱乐部设置

### 2.1 基本信息
- 进入「系统设置」→ 填写俱乐部名称、联系方式、地址
- 配置取消政策：免费取消时间、扣款比例

### 2.2 支付配置
- 进入「微信支付」→ 填写商户号、API密钥、证书序列号
- 点击「测试连接」验证配置正确

### 2.3 通知配置
- 进入「通知模板」→ 配置短信/微信通知模板
- 开启/关闭短信通知、微信通知

## 3. 球馆管理

### 3.1 添加球馆
1. 进入「球馆管理」→ 点击「新增球馆」
2. 填写球馆名称、地址、经纬度
3. 保存后添加场地（场地编号、类型、基础价格）

### 3.2 排期管理
1. 选择球馆 → 选择场地
2. 批量生成14天排期（系统自动）
3. 可手动设置维护时段（暂停预约）

## 4. 活动管理

### 4.1 创建活动
1. 进入「活动管理」→ 点击「创建活动」
2. 选择类型（开放场/培训班/赛事）
3. 选择球馆 → 场地 → 时间段
4. 设置人数上限、价格、取消政策
5. 保存为草稿

### 4.2 发布活动
1. 选择草稿活动 → 点击「发布」
2. 系统自动初始化座位计数器（Redis）
3. 活动状态变为「已发布」，用户可报名

### 4.3 活动模板
1. 创建活动时可「保存为模板」
2. 「活动模板」页面管理模板库
3. 支持「批量发布」：选择模板 → 日期范围 → 星期几 → 批量生成

### 4.4 查看报名
1. 进入活动详情 → 「报名列表」标签
2. 查看已报名人员、签到状态
3. 支持手动签到、取消报名

## 5. 会员管理

### 5.1 查看会员
- 进入「会员管理」→ 查看会员列表
- 支持按姓名、手机号搜索
- 点击会员查看详情（基本信息、会员卡、消费记录）

### 5.2 导入会员
1. 点击「导入」→ 下载Excel模板
2. 填写会员信息（姓名、手机号必填）
3. 上传Excel文件
4. 系统异步导入，显示进度条
5. 导入完成后查看错误报告

### 5.3 会员卡
- 支持次卡、期限卡、储值卡
- 会员详情页可查看/管理会员卡

## 6. 钱包管理

### 6.1 查看钱包
- 进入「钱包管理」→ 查看所有用户钱包
- 查看余额构成（现金余额 + 赠送余额）

### 6.2 充值
- 支持多种充值套餐（充100送10、充200送30等）
- 用户可通过微信支付自助充值
- 管理员可手动调整余额（需权限）

### 6.3 退款管理
- 进入「退款管理」→ 查看待退款列表
- 审核退款申请 → 批准/拒绝
- 批准后自动退回到钱包或发起微信退款

## 7. 场地预约

### 7.1 查看预约
- 进入「场地预约」→ 查看预约列表
- 按日期、场地、状态筛选

### 7.2 取消预约
- 选择预约 → 点击「取消」
- 系统根据取消政策自动计算退款金额

## 8. 数据导出

1. 进入「数据导出」→ 选择导出类型
2. 支持导出：会员列表、报名记录、钱包流水、结算单
3. 系统异步生成 → 完成后下载

## 9. 系统监控

### 9.1 健康检查
- 进入「系统监控」→ 查看各服务状态
- 显示数据库延迟、Redis延迟、内存使用

### 9.2 队列监控
- 查看BullMQ队列状态
- 等待任务数、失败任务数
- 支持重试失败任务

## 10. 人员管理

### 10.1 添加员工
1. 进入「人员管理」→ 点击「添加员工」
2. 选择用户 → 分配角色（前台、财务、教练）
3. 设置权限范围

### 10.2 角色权限
- 平台管理员：全部权限
- 俱乐部管理员：俱乐部相关全部权限
- 前台：活动管理、报名查看、签到
- 财务：钱包管理、退款管理、财务报表

## 常见问题

**Q: 活动发布后还能修改吗？**
A: 已发布且有报名的活动不可修改人数上限，其他信息可修改。

**Q: 用户报名后忘记付款怎么办？**
A: 系统设有付款超时（默认30分钟），超时自动取消并释放座位。

**Q: 如何处理投诉退款？**
A: 进入「退款管理」→ 找到对应订单 → 审核 → 批准退款。

**Q: 座位超卖了怎么办？**
A: 系统使用Redis原子操作+数据库唯一约束双重保障，不会超卖。如遇异常请联系技术支持。
